# openapi-extractor

## Installation

This tool should be added as a dev dependency to the `composer.json` of your app like this:

```json
{
    "repositories": [
        {
            "type": "vcs",
            "url": "https://github.com/nextcloud/openapi-extractor"
        }
    ],
    "require-dev": {
        "nextcloud/openapi-extractor": "dev-main"
    }
}
```
